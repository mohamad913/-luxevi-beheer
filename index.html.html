Hoe werkt de dagelijkse sync?

Elke dag open je Claude via de Chrome extensie op je laptop. Je klikt op ▶ Start Sync op het dashboard.

Claude navigeert dan automatisch naar:
  ① partner.bol.com → haalt LVB-voorraad, bestellingen, ophaalkosten op
  ② MarktMentor → haalt omzet, kosten, winst op

Alle data wordt automatisch in dit systeem opgeslagen. Jij vult enkel de kosten in die niet online staan: transport, inpakmateriaal, andere.

⚠️ Vereist: Claude in Chrome extensie actief + ingelogd op bol.com en MarktMentor
